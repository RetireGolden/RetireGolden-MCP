# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest on `main` | :white_check_mark: |
| Older commits or unmaintained forks | :x: |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report privately via the [Security](https://github.com/RetireGolden/RetireGolden-MCP/security)
tab → **Report a vulnerability**.

Include a clear description, impact, reproduction steps, and any PoC you are
comfortable sharing. We aim to acknowledge reports within **7 business days**.

## Scope

**In scope:** unsafe handling of plan/tool input, injection into tool results,
issues in this repository's server or adapter code, dependency vulnerabilities
that affect the MCP process.

**Out of scope:** incorrect financial or tax calculations (use a regular issue);
client-side AI assistant behavior outside this repo.
