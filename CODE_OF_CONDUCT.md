# Code of Conduct

RetireGolden is a tool people use to make serious decisions about their retirement. We hold the
project's community to the same standard as its math: rigorous, candid, and respectful.

## Expectations

Everyone participating in this project — issues, pull requests, discussions, reviews — agrees to:

- **Be respectful.** Critique code and ideas, never people. No insults, harassment, discrimination,
  or personal attacks, in public or in private channels related to the project.
- **Be candid, kindly.** Direct technical disagreement is welcome and expected; contempt is not.
- **Stay on mission.** Keep discussions relevant to the project. This is not a venue for promoting
  financial products or services.
- **Respect privacy.** Never post someone else's personal or financial information, and be thoughtful
  about sharing your own — plan files and screenshots can contain real financial data.
- **Accept maintainer decisions.** Maintainers make final calls on scope and merges. Advocate,
  disagree, and then let it rest.

## Enforcement

Report unacceptable behavior to [info@retiregolden.org](mailto:info@retiregolden.org). Reports are
reviewed confidentially. Maintainers may edit or remove comments, lock threads, and temporarily or
permanently ban contributors whose behavior violates this code. Egregious behavior (harassment,
threats, doxxing) results in an immediate ban.

Maintainers are held to this code too — including in how enforcement itself is handled.
